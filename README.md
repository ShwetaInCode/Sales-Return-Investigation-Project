# Sales-Return-Investigation-Project

Project Overview
In this Excel nano project, I assume the role of a data analyst for Canadian superstores offering a diverse selection of products in the Furniture, Office Supplies, and Technology categories. The business is currently grappling with an upswing in sales returns, and stakeholders are keen to discern the primary reasons behind these returns.

Objectives
Analyze which Product Categories, Sub-categories, specific Products, Regions, and Managers contribute to the sales returns.
Devise strategies to mitigate these losses.
Predict the potential financial impact six months from now if returns decrease by 50%.
Recommend a Regional Manager for promotion to the position of Head of Sales based on analytical findings.

Tools and Techniques
Power Query: Data import and initial cleaning.
Pivot Tables: Data summarization and analysis.
VLOOKUP: Data lookup and combining datasets.
Excel: An end-to-end project covering all data processing and analysis tasks.

Project Deliverables
Comprehensive data cleaning using outlier detection and missing value imputation.
Detailed descriptive statistics to understand the data.
Insightful data visualizations highlighting key findings.
PowerPoint presentation with concise insights for informed decision-making.
